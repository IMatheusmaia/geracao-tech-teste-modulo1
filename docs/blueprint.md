# **App Name**: GeminiChat

## Core Features:

- Chat Interface: Clean and intuitive chat interface for user interaction.
- Gemini AI Integration: Integrate with the Gemini API for AI-powered chat responses.
- Environment Variables: Load API keys from a .env file using a tool to ensure secure handling of credentials.
- Langchain Integration: Leverage Langchain to orchestrate and manage interactions with the Gemini API.
- Memory Persistence: Save chat history in local storage to preserve context across sessions.
- AI Assistant Persona: Customize the AI assistant's persona and responses for a more engaging conversation.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a professional and calming feel.
- Background color: Light Gray (#F0F2F5), close to white, for a clean interface.
- Accent color: Teal (#009688) to highlight important elements and actions.
- Body font: 'Inter', a sans-serif font, for clear and modern readability.
- Headline font: 'Space Grotesk', sans-serif, to add a techy edge to the main text.
- Use simple, line-based icons for a clean and modern look.
- Clear separation of user and AI messages for easy distinction.
- Subtle animations on message send and receive for user feedback.
